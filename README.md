# Casa Linda


## Pré-Requisitos
- MySQL workbench
- JDK 
- XAMPP
- IntelliJ (preferencialmente)


## Passo a passo
1. Escolher um diretório e descompactar o arquivo backend-casa-linda.zip
2. Abrir o projeto no IntelliJ pelo pom.xml e esperar instalar as dependências do projeto
3. Abrir o XAMPP e dar start no MySQL
4. Abrir o MySQL Workbench e criar um schemma "db_casalinda"
5. Abrir o arquivo LoginApplication.java e executar pelo botão na linha 10
6. Fazer o login com usuario: livia e senha: senha
